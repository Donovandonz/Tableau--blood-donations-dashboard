# Tableau--blood-donations-dashboard
🩸 Blood Donation Tableau dashboard

## Dataset used
-<a href="https://github.com/Donovandonz/Tableau--blood-donations-dashboard/blob/main/blood_donations_state.csv">RAW.blood-donations</a>

## 🎯 Project Overview

This Tableau dashboard project provides comprehensive insights into blood donation patterns across multiple regions, helping blood banks and healthcare organizations optimize their collection strategies and inventory management.

---

## 🛠️ Technologies Used
- **SQL** (MySQL) - Data exploration and analysis
- **Tableau** - Interactive dashboard creation
- **Excel/CSV** - Data cleaning

---

### Key Features
- **Blood Type Analytics**: Detailed breakdown by A, B, AB, O groups
- **Geographic Distribution**: Interactive map of Malaysia and Indonesia
- **Monthly Trends**: Time series analysis of donation patterns
- **State-wise Comparison**: Performance by region
- **Donation Trends Over Time**: Historical pattern analysis

---

## 📊 Tableau Dashboards

### 1. Blood Donations Performance Dashboard
The main dashboard featuring:
- **Blood Flow Insight Cards**: Key metrics for each blood type
- **Donation Trends Over Time**: Line chart with 0-1000K scale
- **Blood Type Composition**: Stacked bar chart by state
- **Monthly Trend Line**: Seasonal pattern analysis
- **Geographic Distribution**: Interactive map of the region

### 2. Blood Group Analytics Dashboard
Detailed blood type analysis showing:
- Blood Type (O): 1,410,284 donations (charts: 0-100K scale)
- Blood Type (B): 895,481 donations (charts: 0-80K scale)
- Blood Type (A): 816,284 donations (charts: 0-60K scale)
- Blood Type (AB): 187,555 donations (charts: 0-15K scale)

## 🔍 Key Insights

### Blood Type Distribution
- **O Positive** is the most common blood type (42.6% of donations)
- **AB** is the rarest (only 5.7% of total)
- **B** type shows strong presence in certain states

### Geographic Patterns
- **W.P. Kuala Lumpur** leads in total donations with 1.19M (35.9% of total)
- **Perak** and **Johor** round out the top 3
- **East Malaysian states** (Sarawak, Sabah) show strong performance
